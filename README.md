# 2026-protocol-optimisation

This notebook enables reproduction of key results from the manuscript from raw sequence data.

Use [uv](https://docs.astral.sh/uv/getting-started/installation/) to install and run the interactive notebook:

```bash
uv run jupyter lab figures.ipynb
```

Optionally uncomment indicated code cells to download FASTQs from climb.ac.uk object storage and perform containment analysis using Skope.
